# Configs

This directory contains the paper release configuration files.

## Preferred Entry Configs

- `llada_twoscale_eval.yaml`
  Main text/code evaluation config for the paper release.
- `lladav_twoscale_eval.yaml`
  Main multimodal evaluation config for the paper release.

## Legacy Compatibility Configs

- `the_new_llada_eval.yaml`
- `lladav_eval.yaml`

These are kept for backward compatibility with the earlier internal naming used during development. For public-facing usage, prefer the two `*_twoscale_eval.yaml` files above.
